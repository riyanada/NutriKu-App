## End-point: Create a Food
### Method: POST
>```
>http://localhost:8080/api/foods
>```

### Body (**raw**)

```json
{
    "nama_makanan": "Air",
    "slug": "air",
    "kalori_kcal": 0,
    "kolestrol_mg": 0,
    "lemak_g": 0,
    "karbohidrat_g": 0,
    "protein_g": 0,
    "kalium_mg": 0,
    "nutrisi": "-",
    "ingredient": "-"
}
```

## End-point: Update a Food
### Method: PUT
>```
>http://localhost:8080/api/foods/2
>```
### Headers

|Content-Type|Value|
|---|---|
|Content-Type|application/json|

### Body (**raw**)

```json
{
    "nama_makanan": "Anggurs",
    "slug": "anggurs",
    "kalori_kcal": 30,
    "kolestrol_mg": 0,
    "lemak_g": 0.2,
    "karbohidrat_g": 6.8,
    "protein_g": 0,
    "kalium_mg": 0,
    "nutrisi": "Vitamin C, Vitamin B1",
    "ingredient": "-"
}
```
